# Smart Incentive Calculator

A web-based calculator to compute sales incentives based on total sales, personal sales, and team structure.

## Features
- Calculates **Incentive Tier**, **Equal Split (70%)**, **Personal 30%**, and **Total Incentive**
- Shows **Projected Incentives** if you or someone else sold a high-value package (AED 1695)
- **Dark Mode** toggle with smooth transitions
- Live **motivational quotes** after each calculation
- 3-digit precision for detailed results

## How to Use
1. Enter **Sales Target**, **Total Sales**, **Personal Sales**, and **Number of Salespersons**
2. Click **Calculate** to see your incentive breakdown
3. Click **Toggle Dark Mode** to switch between light and dark themes

## Notes
- Projected incentive considers the addition of a **AED 1695 package**
- Live quotes fetched from `https://zenquotes.io/`
